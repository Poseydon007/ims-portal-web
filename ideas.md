# IMS Portal Design Ideas

## Response 1 — Austere Government Register
<response>
<text>
**Design Movement:** Swiss International Typographic Style meets Saudi institutional documentation
**Core Principles:** Strict grid, zero decoration, information hierarchy through weight and spacing only, monochromatic with single accent
**Color Philosophy:** Near-black navy (#0E2841) as dominant, warm gold (#C49A28) as the only accent — evoking authority and precision
**Layout Paradigm:** Horizontal rule-driven layout; content flows in clearly separated horizontal bands; no cards with rounded corners
**Signature Elements:** Thick navy header bar, gold rule dividers, monospaced document codes
**Interaction Philosophy:** No animations, instant navigation, document-first
**Animation:** None — static, authoritative
**Typography System:** IBM Plex Mono for codes, IBM Plex Sans for body — technical and precise
</text>

## Response 2 — Executive Dark Portal
<response>
<text>
**Design Movement:** Dark enterprise dashboard — Bloomberg Terminal meets modern SaaS
**Core Principles:** Dark navy background, gold accents, high information density, subtle depth through layering
**Color Philosophy:** Deep navy (#0E2841) background, gold (#C49A28) as primary accent, white text hierarchy
**Layout Paradigm:** Left sidebar navigation + main content area; sidebar collapses on mobile
**Signature Elements:** Glowing gold borders on active states, subtle grid texture on dark backgrounds, monospaced document codes in gold
**Interaction Philosophy:** Hover states reveal gold borders; smooth transitions on route changes
**Animation:** Subtle fade-in on page load, gold border slide-in on hover
**Typography System:** Nunito Sans for UI, Courier New for document codes
</text>

</response>

## Response 3 — Clean Light Corporate (CHOSEN)
<response>
<text>
**Design Movement:** Minimalist corporate intranet — clean, professional, instantly readable
**Core Principles:** Light background, navy text, gold as accent only, generous whitespace, table-driven information
**Color Philosophy:** White/light-gray background for readability, navy (#0E2841) for authority, gold (#C49A28) for hierarchy markers
**Layout Paradigm:** Full-width top nav + content area; category cards in responsive grid; document view uses sidebar + main column
**Signature Elements:** Navy header bar with gold bottom border, gold accent lines on section headings, document code in monospace
**Interaction Philosophy:** Hover reveals gold border on cards; clean breadcrumb navigation
**Animation:** Subtle card lift on hover (translateY -2px), fade-in on page load
**Typography System:** Nunito Sans for all UI, Courier New for codes
</text>
<probability>0.09</probability>
</response>

## CHOSEN: Response 3 — Clean Light Corporate
Rationale: Best matches the document-heavy nature of an IMS portal. Readability and professionalism over visual drama. Gold/navy brand colors are respected throughout.
